## What I'm Building

### [PuppetString](https://github.com/Port53-Cosplay/PuppetString) — Red Team Toolkit for AI Agents

Open-source toolkit for testing AI agent security. Maps to the OWASP Top 10 for Agentic AI (2026). I built it to research the most dangerous and common attacks against AI agents and MCP servers, how they actually work, and what they look like from the defender's side.

- MCP server scanning — dangerous tools, missing auth, input validation gaps
- Workflow fuzzing — tool-abuse, memory poisoning, boundary testing with LLM-judged results
- Indirect prompt injection — steganographic document generation (SVG, HTML, PDF, PNG), encoding bypasses, dynamic payload generation
- Agent-to-agent attacks — trust exploitation, shared memory poisoning, delegation abuse *(in progress)*

Python, Typer, httpx, LiteLLM, Rich, Pydantic, pytest

### [Charon](https://github.com/Port53-Cosplay/Charon) — Job Hunt Intelligence Tool

CLI tool that uses AI to detect ghost job postings, decode toxic-workplace obfuscation, and score opportunities against a personal values profile. Named after the ferryman of the underworld, because job hunting is already hell. I built it because I got tired of decoding "fast-paced environment" and "wear many hats" by hand.

Python, Claude API, SQLite, Typer, Rich

### Raptor Test *(in development)*

Pentesting idle clicker that teaches the engagement lifecycle. Coming soon.
