## What I'm Building

### [PuppetString](https://github.com/Port53-Cosplay/PuppetString) — Red Team Toolkit for AI Agents

Open-source Python toolkit for testing the security of AI agents, agentic workflows, and MCP servers. Maps to the OWASP Top 10 for Agentic AI Applications (2026).

- **MCP server scanning** — discovers dangerous tools, missing auth, input validation gaps
- **Workflow fuzzing** — automated tool-abuse, memory poisoning, and boundary testing with LLM-judged results
- **Indirect prompt injection** — steganographic document generation (SVG, HTML, PDF, PNG), encoding bypasses, goal-based testing with dynamic payload generation
- **Agent-to-agent attacks** — trust exploitation, shared memory poisoning, delegation abuse *(in progress)*

Built with: Python, Typer, httpx, LiteLLM, Rich, Pydantic, pytest
